## Этот проект анализировал набор данных для сравнения контента, созданного ИИ и людьми. Мы начали с загрузки и первичного обзора данных, а также преобразования типов колонок для корректной работы.

## Проведена предобработка: заполнены пропущенные значения в колонке 'notes' и подтверждено отсутствие дубликатов.

## Затем были выявлены и оценены выбросы в числовых показателях, таких как длина текста. Выполнен статистический анализ, включающий описательную статистику, подсчет уникальных значений и корреляционный анализ, который показал сильную связь между длиной текста и количеством слов, а также обратную зависимость между качеством и плагиатом.

## Все ключевые аспекты данных были визуализированы с помощью гистограмм, линейных графиков, столбчатых диаграмм, что позволило наглядно представить распределения и взаимосвязи показателей. Этот комплексный анализ подготовил данные для дальнейших исследований и принятия решений.
